📐 Key Formulas
Arithmetic Mean: r̄ = (1/n) × Σrₜ
Geometric Mean: r_g = [(1+r₁)(1+r₂)···(1+rₙ)]^(1/n) − 1
Variance: σ² = (1/(n−1)) × Σ(rₜ − r̄)²
Std Deviation: σ = √σ²
Risk Premium: RP = r̄_asset − r̄_f

Part D
Excel Extension — Mutual Fund Analysis
Open a blank Excel workbook and follow the steps below. You will build a return analysis model from scratch using a new data set — three Vanguard mutual funds over 2018–2022.

1
Set Up Your Workbook
Layout · Labels · Data Entry
  1. In a blank workbook, type Year in A1, then VTSAX, VBTLX, VGSLX, T-Bill in B1:E1.
  2. Enter the years 2018–2022 in A2:A6.
  3. Enter the return data from the table above in B2:E6. Enter as decimals — e.g., type −0.0517 for −5.17%.
  4. Format B2:E6 as Percentage with 2 decimal places: select the range → Home → Number → Percentage → increase decimals to 2.
✓ Check: Your B2 cell should display −5.17% and contain the value −0.0517.
2
Calculate Arithmetic Mean Returns
AVERAGE function · Row 8
  1. In A8, type the label Arith. Mean.
  2. In B8, enter: =AVERAGE(B2:B6)
  3. Copy B8 across to C8:E8.
  4. Format row 8 as percentage with 2 decimal places.
✓ Check: VTSAX arithmetic mean ≈ 10.55%  |  VBTLX ≈ 0.32%  |  VGSLX ≈ 6.48%
3
Calculate Geometric Mean Returns
GEOMEAN function · Row 9
  1. In A9, type Geo. Mean.
  2. In B9, enter: =GEOMEAN(1+B2,1+B3,1+B4,1+B5,1+B6)−1
  3. Copy across to C9:E9 (adjust column references).
  4. Alternatively, use: =PRODUCT(1+B2:B6)^(1/5)−1 entered as an array formula with Ctrl+Shift+Enter.
✓ Check: VTSAX geo. mean ≈ 8.66%  |  VBTLX ≈ 0.14%  |  VGSLX ≈ 3.71% — all lower than arithmetic means as expected.
4
Calculate Standard Deviation
STDEV.S function · Row 10
  1. In A10, type Std Dev (σ).
  2. In B10, enter: =STDEV.S(B2:B6)
  3. Copy across to C10:E10.
  4. Use STDEV.S (sample), not STDEV.P (population) — we treat these 5 years as a sample.
✓ Check: VTSAX σ ≈ 22.06%  |  VBTLX σ ≈ 8.23%  |  VGSLX σ ≈ 27.65%
5
Calculate Risk Premiums
Excess return above T-Bill · Row 11
  1. In A11, type Risk Premium.
  2. In B11, enter: =B8−E8 (VTSAX mean minus T-Bill mean).
  3. In C11, enter: =C8−E8
  4. In D11, enter: =D8−E8
  5. Leave E11 blank — the T-Bill has no risk premium over itself.
✓ Check: VTSAX risk premium ≈ 9.43%  |  VBTLX ≈ −0.80%  |  VGSLX ≈ 5.36% — VBTLX is negative, meaning bonds barely kept pace with T-Bills over this period.
6
Calculate the Sharpe Ratio
Risk-adjusted performance · Row 12
  1. In A12, type Sharpe Ratio.
  2. In B12, enter: =B11/B10 (risk premium ÷ std dev).
  3. Copy across to C12:D12.
  4. Format row 12 as a number with 2 decimal places (not percentage — Sharpe ratio is unitless).
✓ Check: VTSAX Sharpe ≈ 0.43  |  VBTLX Sharpe ≈ −0.10  |  VGSLX Sharpe ≈ 0.19 — VTSAX offered the best risk-adjusted return over this period.
💬 Discussion Questions — Be prepared to discuss in class
Q1 VGSLX (REIT) had a higher arithmetic mean return than VBTLX, but also a much higher standard deviation. Using your Sharpe ratios, which fund offered better risk-adjusted performance? Does this change your view of REITs as a client holding?
Q2 VBTLX had a negative risk premium over this period. Does that mean bonds are a bad investment? What role might they still play in a diversified portfolio? (Hint: think about what happened in 2018 and 2020.)
Q3 Compare each fund's arithmetic mean to its geometric mean. Which fund shows the largest gap between the two? What does that gap tell you about the volatility of that fund?
